# Termux Cloud Chat
Aplikasi chatting berbasis terminal (Termux).
## Pertama Install
1. pkg update
2. pkg upgrade
3. pkg install git
4. pkg install python
   
Lewati Jika sudah di install

## Install
1. git clone https://github.com/ztncode/ztnproject.git
2. cd chat
3. pip install -r requirements.txt
4. python chat_script.py
